# 👁 THE WATCHER

> *You are the system. You are always watching.*

A surveillance thriller game built for **Flame Game Jam 2026**.

🎮 **[PLAY NOW](https://bader-git-hub.github.io/the-watcher/)**

---

## 🕹️ How to Play

You operate a network of 4 surveillance cameras. Citizens move through each sector. Your job: **identify dissidents before time runs out.**

- **Click a citizen** to open their dossier
- **Wait** — clues reveal themselves over time
- **FLAG** them if suspicious · **CLEAR** them if innocent
- Three wrong calls and **you are replaced**

---

## 🎯 Features

- 4 unique camera feeds — Market District, Residential Block, Transit Hub, Park & Commons
- Procedurally generated citizens with randomized names, jobs, appearances and backstories
- 5 difficulty presets — Easy, Normal, Hard, Expert, and Custom Operation
- Mid-round events — Tip-offs, Blackouts, System Rewinds, VIP Targets
- Quest system with 7 different objective types
- Fully synthesized audio — all sounds and music generated via Web Audio API
- Cinematic UI with CRT scanlines, glitch effects, and surveillance aesthetic
- Volume controls for SFX and music

---

## 🛠️ Built With

- [Flutter](https://flutter.dev/) — UI framework
- [Flame](https://flame-engine.org/) — Game engine
- `dart:js` + Web Audio API — Procedural audio synthesis
- Pure code-drawn pixel art — no external assets

---

## 🎮 Controls

| Action | Input |
|--------|-------|
| Inspect citizen | Click on them |
| Flag as suspicious | ⚑ FLAG button |
| Clear as innocent | ✓ CLEAR button |
| Switch camera | Camera buttons (top left) |
| Pause | ⏸ button (top right) |

---

## 🏆 Scoring

| Action | Points |
|--------|--------|
| Correct flag | +100 × multiplier |
| Correct clear | +20 × multiplier |
| Wrong flag | -50 + strike |
| Missed dissident | -30 + strike |
| Streak bonus | +25 per streak above 2 |
| Mission complete | +500 bonus |
